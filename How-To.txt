## ğŸ§­ Troubleshooting

| Issue | Fix |
|-------|-----|
| **ModuleNotFoundError: No module named 'flask'** | Run `pip install flask` inside your environment |
| **No GPU detected** | The system will fall back to CPU automatically |
| **UI not showing** | Ensure `infant_ui_v6_1.html` is in the same folder as the `.py` file |
| **Memory not clearing** | Refresh the page after clearing memory |
| **Loss always increasing** | Normal for small text snippets; larger datasets improve training |
